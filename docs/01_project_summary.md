# 배움마루 프로젝트 요지

## 프로젝트 정의

배움마루는 복지관, 노인문화센터, 문화센터, 평생교육기관이 회원 관리, 강좌 운영, 수강신청, 추첨, 출석, Excel 출력을 처리하도록 돕는 내부망 업무 시스템이다.

한 대의 Windows 노트북이 임시 호스트가 된다. 호스트 담당자는 해당 PC에서만 열리는 관리 콘솔로 서버와 운영 설정을 관리하고, 접수 직원은 별도 설치 없이 브라우저에서 업무를 수행한다.

> Python 애플리케이션이 SQLite와 내부망 업무 서버를 운영하고, 여러 브라우저가 같은 데이터를 실시간으로 처리하는 시스템

## 해결하려는 문제

- Excel과 종이 신청서를 오가며 생기는 중복 신청, 시간 충돌, 정원 계산 실수
- 여러 직원이 동시에 접수할 때 최신 신청 현황을 맞추기 어려운 문제
- 추첨, 대기자 순번, 출석부, 결과 Excel을 수작업으로 정리하는 부담
- 외부 클라우드에 회원 개인정보를 올리기 어려운 기관 환경

## 프로젝트 목표

- 담당자가 포터블 패키지의 실행 파일로 업무 환경을 열 수 있다.
- 호스트 관리 콘솔은 localhost에서만 접근하고 업무 서버는 명시적으로 시작한다.
- 접수 직원은 내부망 주소와 접속 코드만으로 브라우저에서 업무를 수행한다.
- 회원 정보와 여러 희망 강좌를 한 접수 흐름에서 정확히 저장한다.
- 강좌별 신청, 추첨, 대기, 확정, 출석, Excel 출력까지 한 운영 회차를 처리한다.
- 변경 사항을 접속 중인 다른 직원 화면에 안전하게 반영한다.
- 프로젝트 소유자와 포크 개발자가 Python 코드를 직접 이해하고 검증할 수 있다.

## 제품 경계

포함:

- 내부망 수강신청 업무
- localhost 기반 호스트 관리 콘솔
- 브라우저 기반 직원 화면
- SQLite 기반 로컬 데이터, 백업/복구, Excel 입출력
- Windows 포터블 실행 패키지

제외:

- 외부 공개 수강신청, 결제, 문자 발송, 모바일 앱, 다기관 SaaS
- 프로토타입 단계의 상시 클라우드 운영과 다중 테넌트 관리

## 장기 확장

FastAPI application과 OpenAPI 계약은 Docker, Linux 서버, PostgreSQL, 행사 디스플레이, 자동 백업/알림으로 확장할 수 있게 유지한다. 다만 프로토타입 단계에서는 기관 한 곳의 접수 업무와 Windows 포터블 실행을 먼저 검증한다.
