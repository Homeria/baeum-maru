# 배움마루 라이선스 정책

## 1. 목적

배움마루는 비상업적 목적의 사용, 복사, 수정, 포크, 배포를 허용하는 소스 공개 프로젝트로 운영한다.

프로젝트의 의도는 복지관, 문화센터, 평생교육기관, 개인 학습자가 배움마루를 기반으로 필요한 기능을 개선하고 자유롭게 사용할 수 있게 하는 것이다.

## 2. 적용 라이선스

배움마루는 `PolyForm Noncommercial License 1.0.0`을 따른다.

라이선스 전문은 저장소 루트의 `LICENSE`에서 확인한다.

## 3. 허용하는 사용

- 개인 학습, 연구, 실험, 취미 목적 사용
- 비상업적 기관 또는 비상업적 업무 개선 목적 사용
- 비상업적 목적의 포크, 수정, 재배포
- 배움마루 기반의 개선판 또는 파생 프로젝트 제작

## 4. 허용하지 않는 사용

- 상업적 제품 또는 유료 서비스에 포함하는 사용
- 상업적 이익을 목적으로 한 판매, 호스팅, 재배포
- 라이선스와 출처 고지를 제거한 재배포

## 5. 출처 고지

배움마루를 기반으로 한 포크, 파생 프로젝트, 재배포본은 다음 고지를 유지해야 한다.

- `LICENSE`의 `Required Notice`
- 저장소 루트의 `NOTICE`
- 원 프로젝트 링크: `https://github.com/Homeria/baeum-maru`

이를 통해 사용자와 개발자가 해당 프로젝트가 배움마루를 기반으로 만들어졌음을 확인할 수 있어야 한다.

## 6. 표현 기준

상업적 사용을 제한하기 때문에 배움마루는 OSI 기준의 오픈소스 프로젝트라고 표현하지 않는다.

대신 다음 표현을 사용한다.

- 비상업 소스 공개 프로젝트
- source-available for noncommercial use
- noncommercial source-available project
