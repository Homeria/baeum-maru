# 배움마루 문서 모음

이 문서들은 `배움마루(Baeum-Maru)` 프로젝트를 Codex 또는 다른 개발 도구에서 이어서 진행하기 위한 초기 기획/설계 문서이다.

## 프로젝트 한 줄 요약

**배움마루는 복지관·문화센터·평생교육기관의 수강신청, 추첨, 대기자, 출석부, 엑셀 출력 업무를 내부망에서 처리하기 위한 오픈소스 로컬 호스팅 런처이다.**

## 기본 개발 방향

- Windows 사무용 노트북에서 `exe` 더블클릭으로 실행한다.
- 네이티브 런처 컨트롤 패널에서 서버를 시작/중지한다.
- 같은 내부망에 있는 접수용 노트북들이 브라우저로 접속한다.
- 외부 인터넷 공개는 기본 지원하지 않는다.
- DB는 SQLite를 사용한다.
- 미디어 업로드는 지원하지 않는다.
- 회원, 강좌, 신청, 추첨, 대기자, 출석, 엑셀 입출력에 집중한다.
- 추후 상시 서버 모드 및 Docker 배포로 확장 가능하게 설계한다.
