# Security

배움마루는 로컬 실행형 업무 도구이며, 현재 공개 저장소에는 실사용 데이터가 포함되어서는 안 됩니다.

## 신고

보안 문제를 발견하면 공개 이슈에 재현에 필요한 최소 정보만 작성해 주세요. 실제 회원명, 전화번호, 생년월일, DB 파일, 백업 파일, 엑셀 파일은 첨부하지 않습니다.

## 데이터 취급

- 런타임 DB와 백업 파일은 로컬에만 둡니다.
- 엑셀 출력 파일은 개인정보 파일로 취급합니다.
- 로그에는 업무 처리에 필요한 최소 정보만 남깁니다.
- 외부 포트포워딩이나 인터넷 공개 실행은 기본 사용 시나리오가 아닙니다.
